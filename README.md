# LocadoraGOF
Projeto desenvolvido durante a matéria de programação avançada onde o objetivo é implementar um sistema simples de locadora com todos os padrões que existem no GOF.

- Abstracty Factory
- Factory Method
- Builder
- Prototype
- Singleton


## Estrutura
- Adapter
- Bridge
- Composite

## Comportamento
- Command
- Template Method
- Visitor
